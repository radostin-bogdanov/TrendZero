# Design Guide – Modern UI (2025)

## Visual Style
- Minimalistic, clean, modern.
- Use generous spacing (Tailwind spacing scale 4–10).
- Soft rounded corners (rounded-xl or rounded-2xl).
- Subtle shadows for depth (shadow-sm, shadow-md).
- Color palette uses Tailwind semantic colors, neutral + accent.
- Dark mode first-class.

## Typography
- Use Tailwind typography scale.
- Font pairing: Sans-serif, geometric.
- Headings bold, body regular.
- Max width for text blocks: 65ch.

## Components
- All UI components follow shadcn/ui patterns.
- Use Radix primitives for accessibility.
- Components must be fully responsive.

## Motion
- Micro-animations using Framer Motion.
- Motion should be subtle, 200–300ms.
- Ease-curves: ease-out, ease-in-out.

## Layout
- Mobile-first.
- Grid-based layouts with Tailwind grid utilities.
- Large vertical rhythm.

## Accessibility
- WCAG 2.2 AA.
- ARIA roles.
- Keyboard navigation support.
